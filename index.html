스마트 검사일보 시스템 안내

1. 시스템 개요

  1. 기존의 종이 체크시트를 대체하여, 스마트폰 또는 PC 브라우저에서 바로 입력 가능한 HTML 폼으로 구성되어 있습니다.
  2. 입력된 데이터는 Google 스프레드시트에 실시간 기록되고, 필요 시 Excel 파일로 내려받아 전자결재 시스템과 연동할 수 있습니다.
  3. 이를 통해 현장 검사 → 데이터 수집 → 전자결재에 이르는 전 과정을 디지털화하여, 업무 속도와 투명성을 향상시킵니다.

2. 사용상 팁

  • PC 사용 시에는 브라우저 창 폭을 좁혀 모바일 화면처럼 보시면 입력이 편리합니다.
  • Android 환경에서는 Chrome 브라우저에서 최적화되어 동작합니다.
    • 카카오톡 내장 브라우저로 열면 QR 리더 기능이 제한될 수 있습니다.
  • 카톡 등에서 받은 바로가기를 Chrome으로 열어 홈화면에 추가해두시면 빠르게 접근 가능합니다.
  • PC 환경에서는 주소창 옆 자물쇠 아이콘을 드래그 앤 드롭하여 바탕화면에 바로가기를 생성할 수 있습니다.

3. 등록

  1. HTML 폼을 열고 순서대로 항목을 입력합니다.
  2. 날짜 입력란은 오늘 날짜를 자동으로 가져옵니다.
  3. 이름은 드롭다운에서 선택하거나 직접 입력할 수 있습니다.
  4. 검사 결과 항목은 클릭하여 OK/NG를 선택합니다.
  5. QR 코드 스캔 시 권한 요청이 뜨면 ‘허용’해 주세요.
  6. 일부 폼은 QR 스캔 후 자동으로 시간이 등록되거나 선택 버튼이 활성화됩니다.
  7. 비고란에는 추가로 기록할 내용을 자유롭게 입력합니다.
  8. 제출 버튼은 한 번만 클릭하고, 완료 메시지가 뜰 때까지 약 4초간 기다려 주세요.

4. 조회

  1. 입력한 데이터가 Google 스프레드시트에 정상 등록되었는지 확인합니다.
  2. 오류가 있을 경우 관리자에게 수정 요청을 하시면 빠르게 반영됩니다.

5. 엑셀 다운로드

  1. Excel 파일로 ‘다운로드’ 폴더에 내려받기 됩니다.
  2. PC 및 스마트폰 모두 가능하며, 비밀번호가 설정되어 있습니다.
  3. 시작일자/종료일자는 8자리 숫자(YYYYMMDD)로 입력 후 Enter를 누르면 진행됩니다.
  4. 주기적으로 파일을 다운로드 받아 보관·정리해 두세요.

6. 전자결재

  1. 필요 시 전자문서 결재 기능을 활용합니다.
  2. 결재선은 1차 ~ 3차까지 설정할 수 있습니다.
  3. 결재 신청 시 자동으로 메일이 발송되고, 1~3차 결재자에게 순차 전송됩니다.
  4. 최종 승인 완료 시 신청자에게 승인 완료 메일(PDF 첨부) 이 전송됩니다.
  5. 승인자는 메일 내 링크를 통해 웹창에서 승인 또는 반려 후 창을 닫아주시면 됩니다.
